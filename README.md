# Artemether Drug Chiral Center Analyzer (Python)

This project is implemented in **Python only** as a Streamlit public website for the
**artemether drug**. It visualizes a simplified 3D chiral center for artemether and
lets users switch between **R** and **S** configurations.

## Features

- Python-only web app (`app.py`)
- 2D structure sketch for the Artemether chiral compound
- Interactive 3D model rendered with Plotly
- R/S toggle for stereochemistry learning
- CIP-based explanation for assignment rules

## Run locally

```bash
python3 -m pip install -r requirements.txt
streamlit run app.py
```
